## Jest CLI Flags
* `yarn jest --watch`
  * Run and watch all tests related to the changed file 
* `yarn jest --watchAll`
  * Run and watch all tests
* `yarn jest -t TEST_SUIT_NAME`
  * Run a specific test suit
* `yarn jest --collectCoverage`
  * Run all tests and creates a coverage file
